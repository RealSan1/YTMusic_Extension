Privacy Policy – YouTube Music Synced

Last Updated: June 4, 2025

The "YouTube Music Synced" Chrome extension ("the extension") provides real-time lyrics display and playback control on YouTube Music. We prioritize your privacy, and this policy explains what data we collect and how it is used.

1. Information Collected

The extension collects the following information:

  • Song Information: Title, artist, album name, album art URL, and playback time of the currently playing song on YouTube Music.
  • Playback Status: Whether the song is playing or paused.

This information is collected only from the YouTube Music page (https://music.youtube.com) and used for lyrics display and playback control.

2. Purpose of Use

  • Lyrics Display: Sends song info to the lrclib.net API to fetch synced lyrics.
  • Playback Control: Enables play, pause, seek, and track navigation from the popup UI.
  • User Experience: Displays song info and synced lyrics in the popup to enhance usability.

3. Information Sharing

The extension shares song information (title, artist, album) with the lrclib.net API to request lyrics.

No personally identifiable information (such as name or email) is collected or shared.

Data is not stored on any external server and is handled only within the browser session.

4. Data Storage

Collected data is temporarily stored in browser memory and is not saved to local storage or external servers.

All data is deleted when the browser is closed.

5. User Rights

As no personally identifiable information is collected, data access or deletion requests do not apply.

You may remove the extension at any time through Chrome Settings > Extensions > Remove to stop all data processing.

6. Third-Party Services

Lyrics are provided by lrclib.net. Please refer to their privacy policy on their official website.

YouTube Music data is handled in accordance with YouTube’s policies.

7. Security

The extension collects minimal data and communicates with lrclib.net via secure HTTPS connections.

No external scripts or unnecessary data handling are used.

8. Contact

If you have any questions or concerns, please contact us through the support page on the Chrome Web Store.
